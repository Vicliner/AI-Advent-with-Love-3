# Lesson 5 — Benchmark моделей (Go)

Минимальный пример на Go, который делает запросы к ChatGPT через OpenAI API и показывает результат:

- в консоли (CLI)
- в браузере (локальный web-сервер, ключ не уходит в браузер)

## 1) Подготовка

Нужен ключ OpenAI:

```bash
export OPENAI_API_KEY="YOUR_KEY"
```

## 2) Запуск в консоли

```bash
go run . -mode=cli -prompt "Придумай 5 идей для названия кофейни"
```

## 2.1) Сравнение: без ограничений vs с контролем

Один и тот же запрос отправляется дважды:

- **без ограничений** (обычный запрос)
- **с ограничениями**:
  - явная инструкция по формату ответа
  - ограничение длины (`max_output_tokens`)
  - условие завершения через маркер (обрезается на клиенте; параметр `stop` в Responses API не поддерживается)

```bash
go run . -mode=compare -prompt "Придумай 5 идей для названия кофейни"
```

Настройка контролируемого варианта:

```bash
go run . -mode=compare -control_max_tokens 120 -control_stop "###END###" -prompt "Придумай 5 идей для названия кофейни"
```

Параметры генерации (печатаются в stderr вместе с результатом запроса):

```bash
go run . -mode=cli -temperature 0.7 -top_p 0.9 -max_tokens 200 -prompt "Придумай 5 идей для названия кофейни"
```

Параметр совместимости (на официальный OpenAI API не влияет):

```bash
go run . -mode=cli -top_k 40 -prompt "Придумай 5 идей для названия кофейни"
```

или через stdin:

```bash
echo "Объясни разницу между goroutine и thread" | go run . -mode=cli
```

## 3) Запуск в браузере

```bash
go run . -mode=serve -addr 127.0.0.1:8080
```

Откройте в браузере: http://127.0.0.1:8080

Это чат-интерфейс в стиле ChatGPT со **стримингом** ответа (появляется по мере генерации).

Для задания урока включите в левой панели:

- `control` — один запрос с контролем (формат + длина + stop)
- `compare` — два ответа подряд: **без ограничений** и **с ограничениями**

## 4) Benchmark: слабая/средняя/сильная модель

В веб-интерфейсе есть блок **Benchmark (3 models)**:

- задайте модели (например `gpt-5.2-nano`, `gpt-5.2-mini`, `gpt-5.2`)
- отправьте один и тот же запрос кнопкой **Benchmark**
- получите:
  - время ответа (ms)
  - токены (in/out/total)
  - стоимость (если заполнить Pricing: USD per 1M tokens)
